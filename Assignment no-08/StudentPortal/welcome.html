🎓 Welcome to Student Portal

College Student Information System

Overview

The Student Portal is a web-based application designed to display student information dynamically. This system connects to a database, retrieves student details, and presents them in an organized format.

Key Features

  • Dynamic student data retrieval from database
  • MySQL database integration
  • Responsive web interface
  • Formatted table view of all students
  • Error handling and validation
  • Modern, user-friendly design

Student Information Tracked

  • Student ID (Unique Identifier)
  • Student Name
  • Class/Program
  • Division
  • City
📋 Note: This application uses JSP (Java Server Pages) and JDBC to connect to MySQL database. Ensure MySQL server is running and the database is properly configured before accessing the student list.
